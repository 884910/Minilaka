Struct colonna (in inglese "column" e abbreviato a "col")

E' definita dai campi capacity (pari a 3), size e tower

Documentazione dei campi

◆ size

int col::size

IL campo size può assumere i valori:

  • 0 se la casella è vuota
  • 1 se è presente una sola pedina sulla casella
  • 2 se è presente una torre di due pedine sulla casella
  • 3 se è presente una torre di tre pedine sulla casella

◆ tower

pawn_t* col::tower

Il campo tower ha tipo pawn_t* poichè è un array di pedine che vanno a costituire una torre alta al massimo 3
